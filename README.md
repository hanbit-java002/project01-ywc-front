# zip-doc
ywc-Webapp is a front-end web application for zip-doc. 

## Prerequisites

1. install [nodejs](http://nodejs.org)

1. install gulp-cli

  `npm install --global gulp-cli`

## Usage

```
git clone https://github.com/hanbit-java002/project01-ywc-front.git
cd project01-ywc-front
npm install
gulp
```
